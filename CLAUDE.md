# CLAUDE.md

이 파일은 Claude Code(claude.ai/code)가 이 저장소의 코드 작업을 할 때 가이드를 제공합니다.

## 🚨 필수 개발 규칙 (절대 준수)

### ⛔ 즉시 코딩 금지 원칙 (최우선)
- **파일 생성**: 설정 파일, 구조적 파일, 템플릿 파일 등은 자동으로 생성 가능
- **수정/변경 요청**: 기존 코드 수정, 새로운 기능 추가, 로직 변경 시에는 즉시 코딩 금지
- **피드백 우선**: 수정 지시, 질문, 요청 사항이 있을 때는 먼저 좋은 아이디어, 개선할 부분 등이 포함된 피드백을 제공한다
- **명시적 승인**: 제작자가 확인 후 **'실행' 명령을 하기 전까지는 수정/변경 작업을 하지 않는다**
- **검토 시간 제공**: 사용자가 피드백을 검토하고 결정할 수 있는 충분한 시간 제공

### 🇰🇷 언어 규칙 (필수 사항)
- **모든 대화와 커뮤니케이션은 반드시 한국어로 진행**
- 새로운 작업 창이 열릴 때도 한국어 유지
- 코드 주석도 한국어로 작성
- 변수명과 함수명은 영어 사용 (코딩 컨벤션 준수)
- 문서화는 한국어로 작성
- 한국 투자자 친화적 용어 사용 필수

#### 예외 사항
- 기술적 용어는 영어 그대로 사용 가능 (예: React, TypeScript)
- 라이브러리명, 패키지명은 원본 그대로
- 에러 메시지는 상황에 따라 영어/한국어 혼용 가능

## 프로젝트 개요

이 프로젝트는 한국인을 위한 헝가리어 학습 웹앱입니다. 사용자의 궁극적인 목표는 A1에서 시작해서 B2 수준의 능력을 달성하는 것입니다. 그래서 헝가리어로 설교문을 작성하고 DeepL 번역을 읽고 그 내용의 문제점들을 수정할 수 있는 실제적인 능력을 향상시키는 것에 목표를 둡니다.

## 기술 스택
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Playwright MCP

## 주요 기능
- A1부터 B2까지의 체계적인 헝가리어 학습 커리큘럼
- 개인화된 학습 경로 생성
- 설교문 작성에 특화된 종교 어휘 및 문법
- 게임화된 어휘 학습 (Duolingo, Drops 스타일)
- PDF 학습 자료 업로드 및 통합
- 온라인 상에서 헝가리어 학습에 도움이 되는 자료들을 크롤링 해서 지속적으로 시스템을 업데이트 함
- 진도 추적 및 분석

## 개발 원칙
1. 사용자 피드백을 먼저 제공하고 승인을 받은 후 구현
2. 모든 커뮤니케이션과 문서는 한국어로 작성
3. 단계별 검증 및 품질 관리 필수
4. 헝가리어 학습자의 관점에서 사용자 경험 설계

## 🚨 작업 완료 규칙 (필수 준수)

### ⚡ 모든 작업 완료 시 필수 절차
1. **작업완료 보고서 작성**: `docs/[phase명]-completion-report.md` 파일 생성
   - 완료된 모든 태스크 상세 기록
   - 생성/수정된 파일 목록
   - 기술적 검증 결과
   - 다음 단계 준비 상태 확인

2. **tasks.md 체크박스 업데이트**: `specs/001-hungarian-learning-platform/tasks.md`
   - 완료된 태스크를 `[ ]`에서 `[x]`로 변경
   - 진행률 추적을 위한 정확한 상태 반영

3. **TodoWrite 도구 활용**: 작업 진행 중 실시간 상태 관리
   - 작업 시작 시 'in_progress'로 설정
   - 작업 완료 즉시 'completed'로 변경
   - 한 번에 하나의 태스크만 'in_progress' 상태 유지

### 📋 보고서 작성 기준
- **완료 검증**: 실제 파일 존재 및 기능 동작 확인
- **품질 확인**: 린팅, 타입 체크, 테스트 통과 확인
- **문서화**: 생성된 모든 파일과 설정 기록
- **다음 단계**: 후속 작업을 위한 준비사항 명시

### 🔄 지속적 적용
- **모든 Phase 완료 시**: 해당 Phase 완료 보고서 작성
- **User Story 완료 시**: 개별 User Story 완료 보고서 작성
- **주요 기능 완료 시**: 기능별 완료 보고서 작성

**이 규칙은 프로젝트 전 기간에 걸쳐 예외 없이 적용되며, 작업의 투명성과 추적 가능성을 보장합니다.**
